Overview

Collecting a high-quality dataset is a critical task that demands meticulous attention to detail. To support end-to-end autonomous driving research we collected a dataset of over 2.85 million frames using the CARLA simulation environment for diverse Leaderboard 2.0 challenge scenarios. The dataset supports planning, dynamic object detection, lane divider detection, centerline detection, traffic light recognition, and prediction tasks.

  • Multi-sensor: Camera, BEV, LiDAR-ready design
  • Designed for perception + planning tasks
  • Large-scale and diverse scenarios to address long-tail behaviors
